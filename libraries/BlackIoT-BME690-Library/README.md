# BlackIoT BME690 Library

This Arduino library wraps the [BME690 Sensor API](https://github.com/boschsensortec/BME690_SensorAPI) to provide a simpler experience to use the BME690 Sensor from Bosch Sensortec.

## Supported sensors

- [BME690](https://www.bosch-sensortec.com/products/environmental-sensors/gas-sensors/bme690)
