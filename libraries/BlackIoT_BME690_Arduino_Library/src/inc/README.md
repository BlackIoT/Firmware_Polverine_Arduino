# Install the BOSCH BSEC SDK files for this folder

### From the Bosch BSEC, copy the following files to this folder.

Using a shell:

Where BOSCH_SDK = *The install directory of the downloaded Bosch SDK*

```sh
cp $BOSCH_SDK/algo/bsec_IAQ_Sel/inc/bsec_datatypes.h   bsec_datatypes.h
cp $BOSCH_SDK/algo/bsec_IAQ_Sel/inc/bsec_interface.h   bsec_interface.h
```

Or just drag and drop via a GUI interface.
